# Fases

A atividade contém 15 fases dividida em 4 capítulos:

## Capítulo 1 - Entrada no Sistema
**Objetivo:** Entender bits e construção de números.

### Fase 1 — Inicialização
**Objetivo:** aprender a ligar/desligar bits  
**Desafio:** formar números pequenos (ex: 3)  
**Tipo:** binary-build  

### Fase 2 — Painel de Energia
**Objetivo:** somar potências de 2  
**Desafio:** formar números como 5, 8 e 10  
**Tipo:** decimal-to-binary  

### Fase 3 — Porta de Acesso
**Objetivo:** consolidar binário  
**Desafio:** abrir a porta formando o número correto em binário  
**Tipo:** decimal-to-binary  


## Capítulo 2 - Controle de Dados
**Objetivo:** Converter binário ↔ decimal.

### Fase 4 — Leitura de Dados
**Objetivo:** interpretar números binários  
**Desafio:** converter binário para decimal  
**Tipo:** binary-to-decimal  

### Fase 5 — Escrita de Dados
**Objetivo:** converter decimal para binário  
**Desafio:** representar números dados em binário  
**Tipo:** decimal-to-binary  

### Fase 6 — Sistema Instável
**Objetivo:** alternar entre conversões  
**Desafio:** resolver questões mistas (binário ↔ decimal)  
**Tipo:** mixed-binary  

### Fase 7 — Firewall
**Objetivo:** lidar com valores maiores  
**Desafio:** converter números mais complexos corretamente  
**Tipo:** binary-to-decimal  


## Capítulo 3 - Comunicação Digital
**Objetivo:** Entender representação de caracteres (ASCII).

### Fase 8 — Código Desconhecido
**Objetivo:** associar números a letras  
**Desafio:** converter decimal para caractere  
**Tipo:** ascii-decimal  

### Fase 9 — Tradução Básica
**Objetivo:** reforçar associação  
**Desafio:** converter letra para decimal  
**Tipo:** ascii-reverse  

### Fase 10 — Sinal Corrompido
**Objetivo:** interpretar dados completos  
**Desafio:** converter binário para caractere  
**Tipo:** binary-to-ascii  

### Fase 11 — Mensagem Oculta
**Objetivo:** interpretar sequência de caracteres  
**Desafio:** decodificar uma palavra curta (2 letras)  
**Tipo:** ascii-word  


## Capítulo 4 - Restauração do Sistema
**Objetivo:** Integrar todos os conceitos.

### Fase 12 — Código de Acesso
**Objetivo:** decodificar palavras simples  
**Desafio:** interpretar palavra de 2–3 letras  
**Tipo:** ascii-word  

### Fase 13 — Sistema de Segurança
**Objetivo:** resolver desafios mistos  
**Desafio:** combinar binário e ASCII  
**Tipo:** mixed-ascii  

### Fase 14 — Núcleo do Sistema
**Objetivo:** lidar com maior complexidade  
**Desafio:** decodificar palavra de 3–4 letras  
**Tipo:** ascii-word  

### Fase 15 — Restauração Final
**Objetivo:** aplicar todo o conhecimento  
**Desafio:** decodificar a senha final do sistema  
**Tipo:** final-challenge
